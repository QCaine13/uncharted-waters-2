## 0. 核心灵魂 (North Star)

整个新剧本的设计都为了解决**原作的两个核心痛点**：

| 原作痛点 | 新剧本对策 |
| --- | --- |
| 单人剧情太薄，很快推到后期 | 6 主角从**第一章就互相渗透**，单人路线在第 2-3 章中段接入他人故事 |
| 后期沦为跑商/组舰队/打海盗 | 每个 loop（贸易、战斗、劫掠、投资、探索）都**承担叙事意义**，不是后期填充物 |
| 人和人之间关联弱 | 引入**关系网**系统：同港事件、亲密度、跨线派任务、互为因果 |
| 明处/暗处争斗扁平 | 设计**5 层暗线递进**，从表层反派到文明级真相逐层剥开 |

### 0.1 五层暗线递进（这是与原作最大的区别）

| 层级 | 名义 | 玩家第一眼看到 | 揭开后看到 | 解锁时间 |
| --- | --- | --- | --- | --- |
| L1 | 表层反派 | 海盗王、贪官、敌国战舰 | 单一恶人，可击杀/惩戒 | 第 1 章 |
| L2 | 商业阴谋 | 商会、汇率、特许权、垄断 | 谁在背后买通法官和海关 | 第 2 章 |
| L3 | 政治阴谋 | 列强代理人、刺杀、颠覆 | 谁在资助 L2 | 第 3 章 |
| L4 | 神秘学暗线 | Atlantis / Prester John / 圣者之杖的"传说" | 真实发生过的事件被人掩盖 | 第 4-5 章 |
| L5 | 文明级真相 | L4 背后的世界观真相 | 重新定义"大航海时代"的真正驱动者 | 第 6 章（终局交汇） |

**[决策]** L1-L3 必须是**所有主角都能接触到**的表层叙事；
**[决策]** L4-L5 是**跨线谜题**，需要至少 2 个主角同时推进才能揭开；
**[决策]** 终局时 6 主角在某个**关键节点**（南美 Amazon / 印度洋深处 / 某神秘地点）汇合。

### 0.2 关系网骨架

```
         Joao (冒险)
        /    |    \
  Catalina  Ali  Pietro
   (复仇)  (债/商) (收藏)
        \    |    /
      Ernst --- Otto
      (地图)  (私掠)
```

**[决策]** 关系网用**亲密度数值 (0-100)** + **关系类型 (信任/敌对/暧昧/债主)** 表达；
**[决策]** 同港触发"听到传闻"事件，亲密度到 50+ 解锁"派任务"机制；
**[决策]** 跨线抉择（帮 Joao 找 Atlantis / 还是先帮 Catalina 复仇）会**实质改变**他人剧情走向；

## 1. 原作锚点 + 新增原则

### 1.1 不动的（锚点）

- 时代：15-16 世纪大航海
- 6 主角的**幻想定位**（Joao=冒险 / Catalina=复仇 / Otto=私掠 / Pietro=收藏 / Ernst=制图 / Ali=商债）
- 关键事件节点：Massawa、Zipangu、South America、Amazon 阴谋
- 三大名声：Adventure / Pirate / Trade
- 港口 130 个的位置与名字
- 系统调性：风/洋流/补给/船员/船型选择有意义
- 视觉调性：像素 RPG + 暗色调

### 1.2 加深的（史实深化，方向 A）

- 原作**被一笔带过**的文明补足内部结构：明（朝贡/海禁/士绅/海商）、奥斯曼（米利特制度）、东南亚诸王国（阿瑜陀耶/马六甲/渤泥/满者伯夷）、印度洋（古吉拉特/科罗曼德/孟加拉/锡兰）、东非斯瓦希里、西非马里/桑海余晖、美洲原住民文明（阿兹特克/印加/玛雅余部）
- 给每个大区做**简明档案**（位置/物产/船型/宗教/势力代表人物）

### 1.3 加料的（神秘学，方向 B）

- 保留原作已有的 Atlantis / Prester John / 圣者之杖 / El Dorado 暗线
- 把它**做实**：让"失落的基督教王国"对应东非/拉美某个真实历史影子
- 引入**1 个贯穿 L4-L5 的暗线组织**（具体设定见 §5）
- **[决策]** 暗线组织**不与现实历史组织直接对应**，避免过度影射

### 1.4 不做的

- **不发明完整虚构语言**（吃力不讨好）
- **不引入超自然能力**（不是奇幻 RPG）
- **不重写原作历史大事件**（用户的"原汁原味"边界）
- **不复制原作对话**（避免法律风险，也避免和原作风味冲突）

### 1.5 叙事纪律（政治敏感性指引）

> **元规则**：所有后续 lore 文档都必须遵守；与 [content-methodology.md](content-methodology.md) 配合使用。

#### 五条原则

**P1. 史料为骨，中立为皮**
任何事实性陈述都有考据来源。避免"X 是 Y 的"这种道德审判式表述，改用"X 时代的 Y 现象，史料 A 与 B 的记载存在差异"。

**P2. 多视角呈现，不给上帝判断**
同一事件，从不同势力的 NPC 视角看，呈现不同侧面。玩家自己拼图。剧本作者**永远不当裁判**。

**P3. 描述性语言 > 标签性语言**
- ❌ "奥斯曼 = 压迫者" / "明 = 辉煌无敌" / "原住民 = 受害者" / "欧洲 = 文明传播者"
- ✅ 用"朝贡关系 / 商业网络 / 势力范围 / 当地惯例"这种**描述性**词
- 避开已经被某种话语体系污染的现代国家边界和意识形态标签

**P4. NPC 的偏见 = 角色的一部分**
让 NPC 按其身份和时代说话（90 年代欧洲水手 / 16 世纪奥斯曼官员 / 同时期明朝士绅），玩家要从偏见中**读出真实**。这是文学手法，不是回避立场。

**P5. 史实和价值观，分开写**
- 史实层："X 年，Y 国攻陷 Z 城"——可考据，可写
- 价值观层："X 国是正义的"——不写
- 道德层："Z 城居民很惨"——可以通过 NPC 的反应来呈现，不直说

#### 高风险话题地图（具体怎么处理）

| 话题 | 风险点 | 建议处理 |
| --- | --- | --- |
| 黑奴贸易 | 西方可能淡化，中国/非洲视角可能放大 | 写"贸易存在且残酷"，**不写道德结论**；让不同势力 NPC 各执一词 |
| 殖民主义 | 西方可能赞颂，中国视角批判 | 不用"殖民"二字，改用"势力扩张 / 港口租借 / 商业特许" |
| 明朝海禁 | 中国视角复杂（"自守"还是"压制活力"） | 写**矛盾本身**（官方海禁 vs 民间走私），不下定论 |
| 美洲原住民 | 易"受害者化"或"异域化" | 用**多文明并列**（阿兹特克/印加/玛雅余部都有自己的复杂政治），不简化 |
| 奥斯曼 | 易反派化或神秘化 | 用米利特制度的内部复杂性展示（X 人管 X 人，不是一刀切） |
| 宗教冲突 | 容易现代解读 | 回到 15-16 世纪语境，写"信仰分歧"而非"宗教战争" |
| 莫卧儿 / 印度 | 易"东方风情"化 | 重点写**商业网络**和**政治内斗**，避免风光化 |
| 朝贡体系 | 中国视角可能是"天下秩序"，西方视角是"贸易管制" | 写"互惠贸易的礼仪化"，不站任何一方 |

#### 三条规避操作

- **Op1 信标会做缓冲器**：虚构组织本身体现"不同立场的人聚在一起探索普世遗产"
- **Op2 L5 终局立意天然不偏**："不是谁抢到遗产，而是遗产改变所有人"
- **Op3 6 主角本身就是去标签化的样本**：没有人被写成绝对正确/绝对错误

## 4. 暗线神秘学（L4-L5）

> 这是新剧本的**核心差异点**，但要做到"原作玩家觉得自然"。

### 4.1 已有传说（原作）

- **Atlantis**（亚特兰蒂斯）
- **Prester John**（普雷斯特·约翰，传说中的基督教祭司王）
- **圣者之杖**（Staff of the Saint）—— 原作 Massawa 线
- **El Dorado**（黄金国）
- **黄金面具**（Golden Mask）—— 原作 Pietro 线

### 4.2 我们的扩展逻辑

**核心假设 [决策]**：所有这些传说**指向同一段真实历史**——

> 12-13 世纪，基督教聂斯脱利派（景教）在中亚、印中、东南亚和东非有过相当规模的传教，留下了一些被本地化、混入了佛教/印度教/萨满元素的"基督教王国"。这些王国在 14-15 世纪相继崩溃或被同化（黑死病、蒙古衰退、本地化压力），但留下了**实物、地图、文本**。欧洲大航海时代的"传说"不是空穴来风，而是这些遗存的回声。

**这条暗线让**：
- Prester John = 东非/埃塞俄比亚的真实基督教余脉
- Atlantis = 印尼/印度洋某岛的本地化传说（"消失的岛屿"）
- 圣者之杖 = 聂斯脱利派遗留的传教物
- El Dorado / 黄金面具 = 美洲文明自己的"祖先之地"传说，与基督教传说的重叠

### 4.3 暗线组织

- **[决策]** 引入一个虚构组织作为 L4 暗线的实体：**"信标会"**（Beacon Order）
  - 由 14 世纪末一群葡萄牙水手 + 意大利学者 + 希腊东正教僧侣组成
  - 目的：寻找并保护"普世遗产"（cross-civilization relics）
  - 内部有温和派（开放交流）和极端派（独占/销毁）
  - **与 6 主角都有接触**：Joao 受雇找线索、Catalina 错杀其中一人、Pietro 收藏、Ernst 记录、Ali 资助、Otto 视为竞争对手
- **[决策]** 信标会在 L1 时是"善意之友"，在 L3 后逐渐露出**极端派阴影**，L5 揭示其历史真相
- **[决策]** 信标会的符号、信物、暗号要成为可识别的视觉元素

### 4.4 L5 文明级真相（终局揭示）

**[假设]** 终局时玩家会发现：

> 大航海时代的"扩张"表面是香料/黄金，**深层是多个古老文明的遗存被同时唤醒**——欧洲人、奥斯曼人、明朝人、印度人、东南亚人都在抢同一批"普世遗产"。最终不是谁抢到，而是**这些遗产会改变所有人对世界的理解**。

### 4.5 探索世界：宝物 / 异兽 / 传说事件

> 原则（与 P1-P5 叙事纪律一致）：**尊重传说 + 揭示事实**双重表达；**每件内容都挂到剧情**而非孤立；**不发明幻想生物**；**事件不给超自然归因**。

#### 4.5.1 三大类别（暂定）

| 类别 | 内涵 | 落点文件 | 暗线关系 |
| --- | --- | --- | --- |
| 宝物 / 圣物 | 圣者之杖、金面具、El Dorado 遗物、各文明神物 | `src/data/relicData.ts`（新） | 多数为 L4 信标会追寻的"普世遗产" |
| 珍奇异兽 | 真实存在但被神话化的动物（儒艮→人鱼、海象→海妖、巨龟、犀牛、犎牛） | `src/data/creatureData.ts`（新） | 多数自然探索，Ernst 主线相关 |
| 传说事件 | 暴风雨、海怪目击、鬼船、神秘潮汐、磁暴 | `src/data/legendData.ts`（新） | 表面超自然，**实有自然/人为解释** |

**[决策]** 异兽 = 真实动物 + 神话外壳，不发明新生物。
**[决策]** 传说事件一律给出"自然/人为解释"钩子，不给超自然归因（守 P1）。
**[决策]** 事件**与 questData 区分**——questData 是一次性剧情；legendData 是可重复触发的"风言"。

#### 4.5.2 做厚路径（三个支撑机制）

**机制 1：双重 / 三重表达**

每件宝物 / 异兽 / 事件都有**多层文本**（玩家视角渐进揭示）：

```
【风闻】  在 Cape Verde 以西的某岛，巨大海怪会在月圆之夜浮出海面。
【记录】  1513 年某位葡萄牙领航员称"见到蛇颈巨兽，鸣声让人失神"。
【博物】  18 世纪海洋生物学确认：该岛附近是深海鲸群季节性通道，
          鲸歌次声波在浅海被放大，产生"海怪"幻觉与失神效应。
```

- **风闻**：酒馆/客栈/水手口口相传（首遇）
- **记录**：学者/船长日志/官方文件（亲密度/名声解锁）
- **博物**：博物学家/信标会档案（终态揭示）

**[决策]** 三层文本**均尊重**——风闻是当时的真实认知，博物是现代解释，**不嘲笑**前者。

**机制 2：剧情挂载点**

每件内容都有 `storyHooks` 字段，挂到以下至少一项：

- `protagonist`：谁的故事线会牵涉（6 主角中的 1+ 位）
- `faction`：哪个势力觊觎/封锁/悬赏
- `darkLineLayer`：L1-L5 哪一层
- `fameTrigger`：触发何种名声（Adventure / Pirate / Trade）
- `crossLink`：与谁/何物有剧情因果

**示例**（占位，待 V1 填实）：

| 名称 | 类别 | 主角钩 | 势力钩 | 暗线层 | 因果 |
| --- | --- | --- | --- | --- | --- |
| 圣者之杖 | 圣物 | Joao / Pietro | 葡/明/景教余部 | L4 | Massawa 线的关键物 |
| 美龙涎（抹香鲸） | 异兽 | Ernst / Ali | 葡/荷 | L2 | 香料贸易替代品 |
| Cape Verde 海怪 | 事件 | Catalina（误认） | — | L1 | 与 Catalina 误杀信标会成员相联 |

**机制 3：与现有系统的对接**

- 已有 `data-targets.md` 提了 `discoveryData`——可作为**汇总入口**（指向 relic/creature/legend 三类）
- `questData` 是一次性剧情，**legendData** 是可重复触发的"风言"——两者解耦
- 探索名声（adventure fame）作为**博物层揭示**的奖励
- 港口设施（学者之屋 / 物产馆 / 博物院）作为**博物层文本**的承载点

## 5. 语言策略

### 5.1 不做的

- **不把真实语言塞进对话**——避免知识门槛

### 5.2 做的

1. **三语对照表**（中/英/日）—— 港口名、物品名、NPC 名字、势力名

2. **关键概念双语化**—— 信标会 / Beacon Order / ビコン骑士団
3. **少量造词**—— 仅用于：
   - 信标会的暗号和仪式词（"信标锚" / "海眼" / "风信"）
   - 暗线物品的名字（"圣者之杖" 等沿用原作）
   - 新势力的图腾词（亚齐的"风帆之主"、东非的"海岸之眼"）
4. **港口历史别名**—— 真实历史上曾有的别名作为"行家才知道"的 lore 标签
